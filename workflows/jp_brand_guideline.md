## 1. 브랜드 개요

### 브랜드명
- **영문**: GROSMIMI
- **일문**: グロミミ (グロスミミ X → グロミミ O)
- **발음**: gros의 s는 묵음 → 자연스럽게 「グロミミ」

### 브랜드 의미
- 프랑스어: **gros**(たくさん) + **mimi**(キス)
- 「たくさん、よくキスをする」 = 아기에 대한 사랑 그 자체
- 가장 안전하고 양질의 제품을 전달하겠다는 의지

### 로고
- G 마크 (원형 + 헤드폰 형태) + GROSMIMI 텍스트
- 컬러: 다크 그레이 (#4A4A4A)
- 피드 하단에 작게 배치하는 것이 기본 룰
- 파일: `참고 이미지/이미지.jpg` (로고 원본)

### 브랜드 톤 & 보이스 (일본 시장)
- **따뜻하고 안심감 있는** 톤 (温かくて安心感のある)
- 전문적이되 딱딱하지 않게
- 엄마/파파의 불안을 공감하고 해결해주는 포지션
- 감성 키워드: 安心、やさしい、はじめて、成長、見守る

## 3. 핵심 셀링 포인트 (일본 시장용)

### USP 우선순위
1. **漏れにくい設計** (+CUT / クロスカット) → 엄마들 #1 페인포인트
2. **PPSU 의료 그레이드 소재** → 안전성 강조
3. **洗いやすい** (심플 구조, 식세기 OK) → 일상 편의성
4. **成長に合わせたステップアップ** → 6개월 → 12개월 → 24개월
5. **보온/보냉 6시간** (스테인리스) → 외출 시 편리

### 경쟁 우위
| vs 경쟁사 | Grosmimi 강점 |
|-----------|---------------|
| PP 소재 컵 | PPSU: 내열 200°C, BPA 프리, 변형/변색 없음 |
| 일반 빨대컵 | +CUT 크로스컷: 거꾸로 해도 안 새는 설계 |
| 복잡한 구조 | 심플 파츠: 세척 쉽고 식세기 가능 |
| 고정 핸들 | 360° 회전 핸들: 아기가 편한 각도로 잡기 |

## 4. 일본 타겟 오디언스

### 프라이머리 타겟
- **일본 거주 엄마 (25-38세)**
- 첫째 아이 (6개월~2세) 육아 중
- SNS (인스타, 트위터) 활발 이용
- 제품 안전성에 민감
- 리뷰/비교 콘텐츠 신뢰

### 세컨더리 타겟
- 출산 축하 선물 구매자
- 해외 육아제품에 관심 있는 트렌드세터
- 파파 (최근 일본 육아 참여 증가 트렌드)

### 타겟 페인포인트
1. 빨대컵에서 물이 새서 가방/옷이 젖는 문제
2. 세척이 복잡하고 위생이 걱정됨
3. 어떤 컵을 언제부터 시작해야 하는지 모름
4. PP 소재의 안전성 불안
5. 외출 시 음료 온도 관리
