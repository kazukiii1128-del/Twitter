## 2. 일본 소비자 인사이트 (검색 트렌드 분석)

### 스트로마그 선택 기준 TOP 5 (mybest/mapple 비교 사이트 기반)
1. **漏れない** (안 새는 것) — #1 관심사
2. **洗いやすい** (세척 편의성) — 파츠 수, 식세기 호환
3. **飲みやすい** (아기 음용 편의) — 스트로 연질, 핸들, 무게추
4. **消毒しやすい** (소독 호환) — 열탕/UV/레인지/약액
5. **部品交換** (교체부품 구입 용이) — 스트로, 패킹

### mybest 랭킹 인사이트
- **b.box シッピーカップ: 1위** (20회 흔들어도 0.33mL만 누수)
- 弁(밸브) 있는 스트로 → 거의 안 새는 것으로 검증
- 弁 없는 제품 → 15mL 이상 누수
- **Grosmimi +CUT은 크로스컷 밸브** → b.box보다 우위 가능한 포인트

### 일본 엄마 페인포인트 (Twitter/X 트렌드)
1. **ストローマグ いつから?** — 6개월? 빨리 시작해도 괜찮나? 불안
2. **漏れて鞄がビショビショ** — 가방에서 물 샘 → 외출 스트레스
3. **洗うパーツ多すぎ** — 파츠 많아서 세척 귀찮음
4. **ストロー どれにする** — 선택지 너무 많아 혼란
5. **卒業タイミング** — 언제까지 사용? → 약 2세

### 엄마 커뮤니티에서 버즈(buzz)되는 콘텐츠 유형
1. **비교/ランキング** — "〇〇 vs △△ どっちがいい？"
2. **실패담/공감** — "ストローマグから漏れて大惨事" → 밈 소재
3. **성장기록** — "はじめてストロー飲めた！" 순간 포착
4. **ハック/꿀팁** — 세척 꿀팁, 스트로 교체 시기
5. **선물추천** — 出産祝い로 뭐가 좋을까?

## 4. Grosmimi 일본 차별화 전략

### 공격 포인트 (vs 경쟁사)
1. **PPSU 소재** → PP/Tritan 대비 압도적 안전성·내구성
   - "病院でも使われるメディカルグレード素材"
   - 200°C 내열 → 변형/변색 없음
2. **+CUT 크로스컷** → b.box 슬릿보다 더 확실한 누수 방지
   - "逆さにしても、こぼれない"
3. **스테인리스 라인** → b.box/Richell에 없는 보온보냉
   - "最大6時間 ホット&コールド"
4. **심플 구조** → 세척 편의성
   - "パーツが少ないから、洗うのがラク"

### 방어 포인트 (Grosmimi 약점 보완)
1. **인지도 부족** → 인스타 콘텐츠 양산 + 인플루언서 협업
2. **무게추 없음** → "360°ハンドル"로 잡기 편함을 강조
3. **가격 높음** → "プレミアム素材" 포지셔닝
4. **캐릭터 콜라보 없음** → 심플&스타일리시 브랜딩으로 차별화
